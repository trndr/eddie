eddie
=====

Python3 embedded device identification and exploitation tool kit
```
usage: eddie.py [-h] [--html-port <port>] [--telnet-port <port>]
                [--identify | --exploit | --json-fingerprint]
                ipaddress

Eddie the script kiddie butler is made to assist in identifying embedded
devices

positional arguments:
  ipaddress             IP address of the target

optional arguments:
  -h, --help            show this help message and exit
  --html-port <port>    HTML Port (default: 80)
  --telnet-port <port>  Telnet Port (default: 23)
  --identify
  --exploit
  --json-fingerprint
```
